\begin{abstract}

针对流匹配（Flow Matching）图像编辑模型中的结构保持问题，本文基于近期提出的轨迹正则化编辑框架 FlowAlign，提出了一种轻量级改进方法——FlowLaplacianAlign。原始 FlowAlign 通过对常微分方程（ODE）编辑轨迹施加终端约束，有效提升了编辑结果与源图像之间的一致性。然而，其采用的像素空间点吸引子主要关注局部像素误差，缺乏对多尺度结构信息与频域特征的建模能力。

为此，本文将终端正则项重新构造为隐空间中的多尺度拉普拉斯金字塔（Laplacian Pyramid）势能。该方法在保持原有 ODE 编辑框架不变的前提下，通过分解隐特征中的不同频率残差，实现对全局结构与局部纹理的层级约束。同时，该设计仅依赖基础张量操作，无需引入额外的大规模视觉编码器，因此具有较低的计算与显存开销。

基于 PIE-Bench 标准测试集的实验结果表明，FlowLaplacianAlign 在背景像素一致性指标上略低于原始 FlowAlign，但在 CLIP Score 与 HPS Score 等语义对齐和人类偏好指标上取得了更优表现。实验结果表明，多尺度频域约束能够在结构保持与语义编辑之间提供不同的权衡路径。

综上，FlowLaplacianAlign 提供了一种轻量化的结构正则化方案，并为未来基于频域特征的流式图像编辑研究提供了新的方向。

相关代码放在git仓库：https://github.com/alpha-godzilla/flowalign-laplacian（改自flowalign的git代码：https://github.com/FlowAlign/FlowAlign）

\end{abstract}

\section{引言}
% =========================

近年来，扩散模型与流匹配模型在文本驱动图像生成领域取得了显著进展。然而，在实现目标语义编辑的同时，如何最大程度保留源图像结构一致性，仍然是图像编辑中的核心问题之一。

现有扩散编辑方法通常依赖图像反演过程，通过将真实图像映射至生成模型隐空间后再进行语义修改。这类方法虽然能够获得较强的编辑能力，但往往存在反演误差累积、推理成本较高以及结构漂移等问题。为缓解上述局限，近年来出现了基于流模型（Flow-based Model）的无反演图像编辑方法，通过直接构建编辑轨迹的方式实现源图像到目标图像的连续映射。

FlowAlign 进一步从轨迹正则化视角出发，在编辑 ODE 的终端位置引入结构约束，从而增强了编辑结果与源图像之间的一致性。该方法在 PIE-Bench 等标准基准上取得了较好的结构保持性能。然而，其终端势能函数仍然采用像素空间中的二次误差形式，本质上属于局部像素级约束，对图像中的多尺度结构与频域信息缺乏有效建模。

另一方面，虽然利用 DINOv2 等深层视觉特征可以获得更强的结构表达能力，但这类方法通常需要额外的大规模视觉编码器，并显著增加计算与显存开销，不利于高效部署。

基于上述观察，本文提出 FlowLaplacianAlign。该方法在保持原始 FlowAlign 轨迹修正框架不变的基础上，将终端约束由像素空间误差替换为隐空间中的多尺度拉普拉斯金字塔势能。通过对不同频段残差进行约束，模型能够同时保留全局轮廓与局部纹理结构，并在不引入额外重型网络的情况下实现更高效的结构正则化。

本文的主要贡献如下：

\begin{itemize}
    \item 提出一种基于隐空间拉普拉斯金字塔的终端正则化方法，用于流匹配图像编辑中的结构保持；
    
    \item 在不引入额外视觉编码器的前提下，实现多尺度频域结构约束，保持较低计算复杂度；
    
    \item 在 PIE-Bench 基准上验证了该方法在结构一致性与语义编辑之间的权衡能力。
\end{itemize}

\section{相关工作}
% =========================

\subsection{基于扩散模型的图像编辑}

SDEdit 通过向输入图像注入噪声并重新去噪，实现了基于扩散过程的图像编辑。随后，Prompt-to-Prompt 提出了基于交叉注意力控制的局部语义编辑机制，进一步提升了文本驱动编辑的可控性。InstructPix2Pix 则利用指令监督数据集，实现了自然语言驱动的通用图像编辑。

尽管上述方法具有较强的编辑能力，但大多数方法仍依赖图像反演过程，容易产生误差累积与结构漂移问题。

\subsection{基于流模型的无反演编辑}

Direct Inversion 提出了面向扩散编辑的标准化评测协议，并发布了 PIE-Bench 数据集。随后，FlowEdit 将编辑轨迹建模扩展至预训练流模型中，实现了无反演文本编辑。FlowAlign 在此基础上进一步引入轨迹终端正则化，通过约束编辑轨迹终点提升结构一致性。

本文提出的 FlowLaplacianAlign 延续了该研究方向，并重点关注轻量级多尺度结构约束的构建。

\subsection{流匹配与修正流模型}

Flow Matching 提供了连续时间生成模型的统一训练目标，其核心思想是学习从噪声分布到数据分布之间的速度场。Stable Diffusion 3 基于 Rectified Flow Transformer 架构，实现了高分辨率图像生成与跨模态条件建模。

本文实验均基于 Stable Diffusion 3 Medium 实现。

\subsection{多尺度频域表示}

拉普拉斯金字塔是一种经典的多尺度图像分解方法，可有效分离不同频段中的结构与纹理信息。相比基于深层神经网络特征的结构约束方法，拉普拉斯金字塔仅依赖卷积与池化等基础运算，因此具有更低的计算复杂度。

本文将拉普拉斯金字塔扩展至隐空间特征中，用于构建流编辑轨迹的终端正则化项。

% =========================
\section{方法}
% =========================

\subsection{问题定义}

设输入图像为 $I_{\text{src}}$，其隐空间表示为：

\begin{equation}
z_{\text{src}} = \operatorname{Encoder}(I_{\text{src}})
\end{equation}

给定源文本提示词 $y_{\text{src}}$ 与目标提示词 $y_{\text{tgt}}$，目标是在保持结构一致性的同时，实现符合目标语义的图像编辑。

% =========================
\subsection{FlowAlign 编辑轨迹}
% =========================

FlowAlign 中的编辑轨迹由如下状态定义：

\begin{equation}
q_t = (1-\sigma_t) z_{\text{src}} + \sigma_t \epsilon
\end{equation}

\begin{equation}
p_t = x_t + q_t - z_{\text{src}}
\end{equation}

其中：

\begin{itemize}
    \item $x_t$ 为当前编辑状态；
    \item $\sigma_t$ 为噪声调度；
    \item $\epsilon \sim \mathcal{N}(0, I)$。
\end{itemize}

目标轨迹速度场定义为：

\begin{equation}
v_p =
v_{\theta}(p_t, t, y_{\text{src}})
+
\omega
\left(
v_{\theta}(p_t, t, y_{\text{tgt}})
-
v_{\theta}(p_t, t, y_{\text{src}})
\right)
\end{equation}

其中 $\omega$ 表示 classifier-free guidance 系数。

对应的源轨迹速度场为：

\begin{equation}
v_q = v_{\theta}(q_t, t, y_{\text{src}})
\end{equation}

% =========================
\subsection{隐空间拉普拉斯终端约束}
% =========================

原始 FlowAlign 采用像素级终端约束：

\begin{equation}
m_{\text{pix}}
=
\frac{1}{2}
\|
\hat{p}_0 - \hat{q}_0
\|_2^2
\end{equation}

该形式主要关注局部像素误差，对多尺度结构信息缺乏建模能力。

为此，本文提出隐空间拉普拉斯终端约束。首先，对预测终点隐特征构建拉普拉斯金字塔：

\begin{equation}
\mathcal{L}_l(\cdot)
\end{equation}

表示第 $l$ 层频率残差。

最终终端势能定义为：

\begin{equation}
m_{\text{lap}}
=
\frac{1}{2}
\sum_{l=0}^{L}
\left\|
\mathcal{L}_l(\hat{z}_p)
-
\mathcal{L}_l(\hat{z}_q)
\right\|_2^2
\end{equation}

其中：

\begin{itemize}
    \item $\hat{z}_p$ 为目标轨迹预测终点；
    \item $\hat{z}_q$ 为源轨迹预测终点；
    \item $L$ 为金字塔层数。
\end{itemize}

对应修正速度项为：

\begin{equation}
v_{\text{modify}}
=
-
\gamma
\nabla_{\hat{z}_p}
m_{\text{lap}}
\end{equation}

最终更新公式如下：

\begin{equation}
x_{t+\Delta t}
=
x_t
+
(\sigma_{t+1}-\sigma_t)
(v_p-v_q)
+
v_{\text{modify}}
\end{equation}

% =========================
\subsection{算法流程}
% =========================

\begin{algorithm}[t]
\caption{FlowLaplacianAlign}
\label{alg:flowlaplacian}
\begin{algorithmic}[1]

\State Encode source image:
\[
z_{\text{src}}
\leftarrow
\operatorname{Encoder}(I_{\text{src}})
\]

\State Initialize:
\[
x_0 \leftarrow z_{\text{src}}
\]

\For{each timestep $t_i$}

    \State Compute $q_{t_i}$ and $p_{t_i}$

    \State Estimate velocity fields:
    \[
    v_p, v_q
    \]

    \State Predict terminal states:
    \[
    \hat{z}_p,\hat{z}_q
    \]

    \State Compute Laplacian regularization:
    \[
    v_{\text{modify}}
    \leftarrow
    -
    \gamma
    \nabla m_{\text{lap}}
    \]

    \State Euler update:
    \[
    x_{t_{i+1}}
    \leftarrow
    x_{t_i}
    +
    \Delta t
    (v_p-v_q+v_{\text{modify}})
    \]

\EndFor

\State Decode final latent:
\[
I_{\text{edit}}
\leftarrow
\operatorname{Decoder}(x_{t_{\text{end}}})
\]

\end{algorithmic}
\end{algorithm}

\section{实验}
% =========================

\subsection{实验设置}

本文采用 PIE-Bench 作为评测基准。该数据集包含 700 个图像编辑样本，并提供编辑区域掩码。所有背景一致性指标均仅在非编辑区域计算。

实验基于 HuggingFace Diffusers 框架实现，后端模型采用 Stable Diffusion 3 Medium。

除终端约束形式外，其余超参数均与 FlowAlign 保持一致，包括：

\begin{itemize}
    \item CFG scale = 13.5
    \item regularization weight $\gamma=0.01$
    \item inference steps = 50
\end{itemize}

\subsection{结果分析}
% =========================

实验结果表明，FlowAlign 在 MSE、PSNR 与 SSIM 等背景一致性指标上表现更优，说明像素级终端约束能够更有效地保持局部像素一致性。

相比之下，FlowLaplacianAlign 在 CLIP Score 与 HPS Score 上取得更高结果，说明多尺度频域约束能够为目标语义提供更大的生成自由度，从而增强文本语义融合能力。

此外，FlowLaplacianAlign 不依赖额外视觉编码器，仅通过基础张量运算即可完成终端约束，因此具有较低的额外计算成本。

整体而言，该方法并非完全替代像素级约束，而是在结构保持与语义编辑之间提供了另一种具有不同偏好的正则化路径。

% =========================
\section{结论}
% =========================

本文提出了一种用于流匹配图像编辑的轻量级终端正则化方法 FlowLaplacianAlign。该方法通过在隐空间中引入多尺度拉普拉斯金字塔约束，实现了对图像结构与纹理信息的频域建模。

实验结果表明，该方法能够在保持较低计算复杂度的同时，提升编辑结果的语义对齐能力，并在结构一致性与语义表达之间形成新的权衡。

未来工作将进一步探索：

\begin{itemize}
    \item 不同频段的自适应权重设计；
    \item 与大型视觉特征的混合约束；
    \item 面向视频编辑的时序频域一致性建模。
\end{itemize}
